Projects

Internet of Robotic Things

This project focuses on bringing a robot onto the Internet of Things (IoT). Using an on-board Raspberry Pi, the robot transmits lidar data, battery level, and other telemetry to the Pi. The Pi then uses wifi to transmit that data to an online dashboard called Thingsboard. A full report of the project so far can be downloaded here.

Autonomous Indoor Navigation

This is my graduate project which focuses on using the lidar to navigate through indoor environments. This is to be done using a residual convolutional neural network. The network works upon what the robot sees and where the robot wants to go, much like how humans navigate. This is done using lidar data and a relative target position. Once the network is trained, it will take that data and produce translational and rotational velocities which the robot will translate into motor power. The neural network is trained using publically available datasets and I am working on a way to get more training data by collecting data on the Pi from the robot.

Kepler Orbital Simulator

This was my undergraduate project where I developed the back-end for an orbital simulator using Keplers Laws of oribital mechanics. This simulator displays relative planetary positions based on any date. This project was motivated by a need for new software in the Appalachian State Astronomy Department. Lab students were using a simulator that was older than I was, and so it was slow, glitchy, and difficult to use. A collegue and I decided to update it to the 21st century.
