about the site:

This site was made as a personal project to tinker with a Raspberry Pi(right). The Pi is a small inexpensive computer on a chip that's perfect for a web server (and many other applications).

A big thanks to Pablo Puigcerber's step-by-step tutorial on how to set up the Pi for a webserver. The domain name was procured with Namecheap, an informative and easy-to-use site to purchase any domain for cheap. The bulb icon on the tab was made by Freepik from www.flaticon.com and is

The site itself was made from scratch using what I remember
from a client-side web development class I took in undergrad.
