projects

robot guide dog (in progress)

The goal of my Master's project is to design a robotic guide for the blind and visually impaired. To do this, the robot is equipped with a Light Detection and Ranging (LiDAR) device to detect objects around it.

The plan is to add a handle to the robot, and have the robot guide the user and avoid obstacles. If faced with an intersection, the robot should stop and ask the user which way they would like to go.

kepler orbital simulator

This simulator was made as my senior capstone project for the Appalachian State Astronomy Labs. This program has the ability to display planetary positions given any date using Kepler's laws of planetary motion.

The simulator that the lab students were using before was made before I was born. It was slow, glitchy, and difficult to use. A collegue and I decided to update it to the 21st century. Here is a video of the simulator in action.
